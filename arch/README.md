# Arch Linux Scripts

## arch_setup

Run:

```
$ curl --silent --output arch_setup https://raw.githubusercontent.com/rvenutolo/Scripts/main/arch/arch_setup \
  && bash arch_setup
```

Notes:

* Double-check `packages/*.txt` files to ensure these are the desired packages.
